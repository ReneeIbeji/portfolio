My Corne Keyboard

A CRKBD (Corne Keyboard) is an erganomic mechanical keyboard with an open source design

Mine is the V3 hostswappable version, built from a kit brought from mechboards, it was my first time soldering since secondary school

The keyboard is flashed with a modified version of the qmk firmware for the crkbd by mechboards (written in C++) (link here) to clear the left oled screen to make sure there isn't overlapping text for input info, Adds my cute logo to the right oled screen and make some changes to the keymap (eg adding volume controls)

The keys next to the space and enter keys (the middle keys of the bottom section which are pressed by the thumb) can be held down to switch between the layers, allowing one key to have multiple functions depending on which one is held down.
